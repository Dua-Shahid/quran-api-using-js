# 📖 Quran API Using JavaScript

A simple and responsive web application that fetches and displays Quran data using the AlQuran Cloud API.

## 🚀 How It Works

* Sends a secure HTTPS GET request to the AlQuran Cloud API using the JavaScript Fetch API.
* Receives the response in JSON format.
* Extracts the Surah name, English name, English translation, and revelation type.
* Displays the data dynamically on the webpage using JavaScript.

## ✨ Features

* Fetch Quran data from the AlQuran Cloud API
* Display Surah information dynamically
* Responsive user interface
* Simple and clean design

## 🛠️ Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* JavaScript (ES6)
* Fetch API
* REST API
* JSON
